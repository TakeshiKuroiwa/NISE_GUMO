## ⚡ クイックスタート

### 前提条件

- Node.js 20+
- Docker（オプション）

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/username/repo.git
cd repo

# 依存関係をインストール
npm install

# 環境変数を設定
cp .env.example .env
# .env を編集する

# 開発サーバーを起動
npm run dev
```

ブラウザで http://localhost:3000 を開く。
